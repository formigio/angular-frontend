Formigio

Commitment based Goals and Tasks.

Features

Login / Sign Up
  • Google Account Login
  • Email/Password Login
Teams
  • Add Teams
  • Invite people to Teams
Goals
  • Add Goals to Teams
  • Edit Goal Description
  • Mark Goal as Accomplished
  • Filter Accomplished Team Goals
Tasks
  • Add Tasks for Goals
  • Set the Task Work Status (Started, Paused, Blocked, Complete)
Commitments
  • Commit to Tasks choosing when and for how long you will work on it
  • Navigate to Commitments for a Specific Date
  • Manage Commitments
  • Filter Commitments by Future Date
  • Filter Commitments by Task Completed
